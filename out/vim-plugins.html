Gitsigns

  • ]c - jump to next hunk
  • [c - jump to previous hunk
  • Space + gshs - stage hunk
  • Space + gshr - reset hunk
  • Space + gshS - stage all hunks in current buffer
  • Space + gshu - undo last hunk
  • Space + gshR - reset all hunk in current buffer
  • Space + gshp - preview hunk

  • Space + gsb - git blame (short)
  • Space + gsB - git blame (full)
  • Space + gst - toggle deleted
  • Space + gsx - trouble

Search and Replace

  • Space + rs - selections
  • Space + ro - open
  • Space + rw - replace current word
  • Space + re - replace current expression
  • Space + rf - replace current file
  • Space + rW - replace current WORD
  • Space + rF - replace in files via Spectre
Tip: To replace in all open buffers Space + rb can be used:

 Space + rbs - selections
 Space + rbo - open
 Space + rbw - replace current word
 Space + rbe - replace current expression
 Space + rbf - replace current file
 Space + rbW - replace current WORD
