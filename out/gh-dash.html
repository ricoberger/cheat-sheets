Movement

  • k - move up
  • j - move down
  • h - previous section
  • l - next section
  • g - first item
  • G - last item
  • Ctrl+d - preview page down
  • Ctrl+u - preview page up

Refresh, Open, Copy

  • r - refresh
  • R - refresh all
  • s - switch view
  • p - open in Preview
  • o - open in GitHub
  • n - open in Neovim
  • y - copy number
  • Y - copy url
  • / - search

Actions

  • a - assign
  • A - unassign
  • c - comment
  • d - diff
  • C - checkout
  • x - close
  • w - ready for review
  • X - reopen
  • m - merge

Misc

  • ? - help
  • q - quit
